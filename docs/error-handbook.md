# Claude Code 错误处理手册

本文档列出 Claude Code 使用过程中可能遇到的已知错误及其解决方案。

---

## 1. invalid_request_error — tool result not found

**错误信息：**
```
400 {"type":"error","error":{"type":"invalid_request_error",
"message":"invalid params, tool result's tool id not found"}}
```

**原因：** 对话上下文被压缩时，工具调用的引用与系统状态失去同步。

**解决方案：**

- 重新执行相同的操作（通常一次成功）
- 如果频繁出现，尝试用 `/clear` 清空会话重新开始
- 将复杂任务拆分成更小的步骤

**用户可做：**
- 不需要做任何代码修改，遇到时重试即可

---

## 2. 上下文长度超限 (Context Limit Exceeded)

**错误信息：** 系统提示上下文已达到上限，无法继续对话。

**原因：** 对话历史过长，超过了模型的上下文窗口。

**解决方案：**

- 开始新的对话
- 使用 `/clear` 清空当前会话状态
- 将大型任务拆分成多个独立会话

**用户可做：**
- 定期开启新会话，避免单一对话过长

---

## 3. MCP Server 连接错误

**错误信息：** MCP server 无响应或连接超时。

**原因：** MCP 服务器未运行、端口冲突、或配置错误。

**解决方案：**

- 检查 MCP server 是否正在运行
- 验证 `settings.json` 中的 MCP 配置
- 重启 Claude Code

---

## 4. 工具执行超时

**错误信息：** 某个工具（如 `Bash`、`Read`）执行超时。

**原因：** 命令耗时过长或系统资源不足。

**解决方案：**

- 检查命令本身是否正常
- 增加超时时间（如 `timeout` 参数）
- 将长时间任务改为后台运行

---

## 5. 权限被拒绝 (Permission Denied)

**错误信息：** 工具调用提示权限不足。

**原因：** Claude Code 未获得执行该操作的权限（如修改文件、运行命令等）。

**解决方案：**

- 检查 Claude Code 的权限设置
- 在系统偏好设置中确认 Claude Code 已被允许访问相关功能
- 用户需手动授权

---

> **通用原则：** 如果遇到不在此列表中的错误，尝试 `/clear` 后重试，或重新启动 Claude Code。
